Who is this platform for?

This shapes the whole architecture — pick what fits best

A

Closed Friend Group

Invite-only. You control who's in. Ratings and profiles only visible to members. Like a private Discord but for media opinions.

B

Semi-Public

Anyone can sign up. Users choose what's public vs private. Friend groups still exist for friend-specific data views.

C

Fully Public

IMDB-style. Everything visible to anyone. Profiles, ratings, watchlists all public by default. Maximum discoverability.
